# 7z

## compress & encrypt

```bash
file_org="abc.txt"

LC_ALL=C tr -dc '[:alnum:]' < /dev/urandom | fold -w 20 | head -n1 > pass.txt
pass=$(cat pass.txt)
```

```bash
7z a -p"${pass}" -t7z -ssw -m0=LZMA2 -mhe=on "${file_org}.7z" "${file_org}"
or
7z a -p"${pass}" -mhe=on "${file_org}.7z" "${file_org}"
```

## list archive contents

```bash
7z l -lst ${file_org}.7z
```

## extract

```bash
7z e "${file_org}.7z"
```
