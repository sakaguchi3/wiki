# python uv

## install

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

[doc](https://docs.astral.sh/uv/#highlights)

## command

```bash
uv python install 3.10 3.11 3.12
uv python list

uv add <soft>
```

## init

```bash
uv init <project>
cd <project>

uv run hello.py

. .venv/bin/activate
deactivate

uv lock
```
