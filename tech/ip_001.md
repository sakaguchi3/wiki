# ipコマンド

|net-tools| 	iproute2|
|:----:|:----:|
|ifconfig| 	ip l (ip link)|
|ifconfig -a| 	ip a show (ip addr show)|
|ifconfig eth0 up| 	ip link set eth0 up|
|netstat| 	ss|
|netstat -i| 	ip -s link|
|netstat -l |	ss -l|
|netstat -r |	ip r (ip route)|
|route [add｜del] |	ip route [add｜del]|
|route -n |	ip route show|
|arp -n |	ip n (ip neighbor)|
