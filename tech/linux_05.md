# Linux - 監視コマンド


### コマンド

| コマンド | 機能 | |
|--|--|--|
|vmstat 2  | cpu, memory, disk-io などを監視   |   |
|dstat 2  | vmstatの拡張版．トラフィックの監視とかもできる．  |   |
|sar  |過去の負荷の履歴   |   |
