# mongodbのインストール @mac



## install
 
```bash
$ brew install mongodb
```


## start/stop 


```bash
$ brew services start mongodb
$ brew services stop mongodb
```

起動プロセス一覧

```bash
$ brew  services list
```



## 接続


```bash
$ mongo
> show dbs;
```
