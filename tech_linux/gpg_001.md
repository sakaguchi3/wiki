# gpg - ファイル暗号


### ファイル暗号化


```bash

 $ gpg -c hoge.dummy
 $ passwd: 
```


### ファイル復号化

```bash
$ gpg hoge.dummy.gpg
$ passwd:
```

### パスワードファイルから暗号


```bash
# 暗号
gpg --batch --passphrase-fd 0  -c hoge.dumy < pass.txt 
# 復号
gpg --passphrase-fd 0  hoge.dumy.gpg < pass.txt
```
