# jq


### バッククオートを解除する

```bash
$ cat json-tmp01.json \
    | jq '.seatbid[0].bid[0].adm | fromjson'    \

$ cat json-tmp01.json \
    | jq '.seatbid[].bid[].adm | fromjson'    \
``` 


### json -> formatted_json , Add indent

```bash
$ jq . json-tmp.json 
```


### formatted_json -> json , Remove indent

```bash
$ jq . json-tmp2.json -c
```
